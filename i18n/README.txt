A repository for the .po files containing translations for silva.
Naming pattern should be silva-<country-code>.po

There's also a silva.pot file containing the default msgids.

Please consider helping us translate silva into your language, or perfecting
an existing translation. Translations can be edited online, one string at
a time. Silva's translations are found here:

https://launchpad.net/products/silva/+series/main/+pots/silva/
